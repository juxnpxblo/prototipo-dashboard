## Installation

1. Clone the repo and enter its folder
```sh
git clone https://github.com/juxnpxblo/prototipo-dashboard.git
cd prototipo-dashboard
```
2. Install packages
```sh
npm install
```
3. Start the server
```sh
npm start
```
